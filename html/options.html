User-Agent 007

Tips:

Tip: Enter * (an asterisk) into the domain field to enable a particular user-agent across all sites that you didn't assign a user-agent to.

Note #1: Do not use subdomains or add "www". Adding the root domain will automatically enable that particular user agent across all of its subdomains.

Note #2: If the user agent didn't change immediately on refresh, refresh the page again and it should work.
